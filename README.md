# Claude Prompts

This project is a library of prompts that can be used to create agents. The focus is on Claude Code, but you can use them with other AI agents as well.

## Prompts

| File | Description |
|---|---|
| [pencil-design-agent.md](prompts/pencil-design-agent.md) | Agent for creating app designs in Pencil via MCP, with multi-file design system management and bash-based cross-file search. |

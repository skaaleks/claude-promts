# Claude Prompts

This project is a library of prompts that can be used to create agents. The focus is on Claude Code, but you can use them with other AI agents as well.

## Prompts

- [design-system-generator](prompts/design-system-generator.md) — Generates a complete design system specification as a structured markdown file (colors, typography, spacing, shadows, breakpoints, etc.)
- [ui-ux-design-agent](prompts/ui-ux-design-agent.md) — Designs structured UI/UX screens with full user scenario flows, state variations, reusable components, and canvas layout management.
- [spa-frontend-developer-agent](prompts/spa-frontend-developer-agent.md) — SPA frontend development agent for Next.js + Tailwind projects — pure UI layer with client-side rendering, backend API integration, and design quality standards.
- [playwright-mcp-qa-agent](prompts/playwright-mcp-qa-agent.md) — Black-box QA testing agent that uses Playwright MCP to interact with web apps through a real browser, executing user scenarios and producing structured bug reports.
